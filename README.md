Isso é tão simples que provavelmente alguém já fez antes, mas como eu não achei, resolvi fazer meu próprio pacote e postar no GitHub. Um .Deb pra instalar o Brasfoot e fazer ele rodar "nativamente" no Linux via java (já que o Brasfoot é produzido em java). Se estiver desconfiado pode baixar as Pastas Debian e usr, olhar os arquivos, e compilar você mesmo o seu .Deb!
